# 📖 Project Overview

This project presents a comparative analysis of **TikTok** and **Instagram** advertising effectiveness among undergraduate students in Malaysia using **Power BI**.

The analysis follows the **AIDA Model (Attention, Interest, Desire, Action)** to evaluate how advertisements on each platform influence consumer behaviour.

A survey consisting of **200 respondents** was collected using Google Forms and analysed through Power BI dashboards.

---

# 🎯 Objectives

This study aims to:

- Compare TikTok and Instagram advertising effectiveness.
- Evaluate consumer behaviour using the AIDA Model.
- Analyse demographic characteristics of respondents.
- Identify which platform performs better at each AIDA stage.
- Provide marketing recommendations for businesses targeting Malaysian undergraduate students.

---

# 🧠 AIDA Framework

| Stage | Description |
|--------|-------------|
| 👀 Attention | Ability of advertisements to attract attention |
| ❤️ Interest | Ability to sustain user interest |
| ⭐ Desire | Ability to create purchase desire |
| 🛒 Action | Ability to encourage purchase intention |

The Power BI dashboard provides an interactive comparison between TikTok and Instagram advertising performance using demographic analysis, AIDA scores and correlation analysis.

# 📈 Dashboard 1 — Respondent Demographics

<img src="Images/demographic_dashboard.png" width="900">

### Key Findings

- 200 respondents participated.
- Female: 52%
- Male: 48%
- Majority aged 21–23 years old.
- Instagram is preferred by 70.5% of respondents.
- Most respondents spend 3–5 hours daily on social media.

---

# 📱 Dashboard 2 — TikTok Performance

<img src="Images/tiktok_dashboard.png" width="900">

### Highlights

Overall AIDA Score:

**3.50**

Average Scores

| Stage | Score |
|--------|------|
| Attention | 3.46 |
| Interest | 3.41 |
| Desire | 3.58 |
| Action | 3.57 |

### Interpretation

TikTok performs particularly well in generating purchase desire and encouraging action.

---

# 📷 Dashboard 3 — Instagram Performance

<img src="Images/instagram_dashboard.png" width="900">

### Highlights

Overall AIDA Score

**3.51**

Average Scores

| Stage | Score |
|--------|------|
| Attention | 3.64 |
| Interest | 3.55 |
| Desire | 3.46 |
| Action | 3.38 |

### Interpretation

Instagram performs strongly in attracting attention and maintaining audience interest.

---

# 🔥 Dashboard 4 — Comparison

<img src="Images/comparison_dashboard.png" width="900">

### Platform Comparison

| AIDA Stage | TikTok | Instagram |
|------------|---------|------------|
| Attention | 3.46 | **3.64** |
| Interest | 3.41 | **3.55** |
| Desire | **3.58** | 3.46 |
| Action | **3.57** | 3.38 |

### Key Insight

Instagram performs better in the early stages of the customer journey.

TikTok performs better in encouraging purchase intention and action.

---

# 📉 Dashboard 5 — TikTok Correlation Matrix

<img src="Images/correlation_tiktok.png" width="900">

### Strongest Relationship

Desire ↔ Action

**r = 0.85**

This indicates that stronger purchase desire leads to higher purchasing behaviour on TikTok.

---

# 📉 Dashboard 6 — Instagram Correlation Matrix

<img src="Images/correlation_instagram.png" width="900">

### Strongest Relationship

Interest ↔ Desire

**r = 0.78**

This indicates that maintaining user interest is important for creating purchase desire on Instagram.

---

# 💡 Key Findings

✅ Instagram performs better in

- Attention
- Interest

✅ TikTok performs better in

- Desire
- Action

✅ Both platforms receive positive overall evaluations.

✅ Instagram is suitable for brand awareness.

✅ TikTok is suitable for increasing conversions.

---

# 📌 Business Recommendations

Businesses targeting undergraduate students should:

- Use Instagram for branding and awareness campaigns.
- Use TikTok for conversion-focused marketing.
- Develop platform-specific advertising strategies.
- Combine both platforms for maximum campaign effectiveness.
